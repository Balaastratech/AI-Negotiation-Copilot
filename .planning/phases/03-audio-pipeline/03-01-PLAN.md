<objective>
Implement the core AudioWorklet processors required by the Gemini Live API for raw PCM capture and playback.

Purpose: The browser's standard MediaRecorder API produces WebM/Opus, which Gemini rejects. These worklets are strictly necessary to perform the raw Float32 <-> Int16 conversions at the correct sample rates (16kHz for input, 24kHz for output).
Output: Two JavaScript files in `frontend/public/worklets/` that can be loaded directly by the browser's AudioContext.
</objective>
